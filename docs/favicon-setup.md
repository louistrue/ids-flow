# Favicon Setup Guide

This guide explains how to set up favicons for IDSedit using the icon8 "stacked organizational chart highlighted first node" icon.

## Step 1: Download the Icon

1. Visit [Icons8](https://icons8.com) and search for "stacked organizational chart highlighted first node"
2. Alternatively, search for "organizational chart" and look for the stacked/hierarchical version
3. Download the icon as PNG format, preferably 512x512 pixels or larger
4. Save it as `public/icon-source.png` in this project

## Step 2: Generate Favicons

Once you have the source icon, run:

```bash
npx tsx scripts/generate-favicons.ts public/icon-source.png
```

This will generate all the required favicon files:
- `favicon.ico` (multi-size ICO file)
- `favicon.svg` (SVG version)
- `favicon-16x16.png`
- `favicon-32x32.png`
- `favicon-48x48.png`
- `apple-touch-icon.png` (180x180)
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`
- `mstile-150x150.png` (Windows tile)

## Step 3: Verify

The favicons are already configured in `app/layout.tsx`. After generating the files, restart your dev server and check:

- Browser tab icon
- Browser bookmarks
- Mobile home screen (iOS/Android)
- Windows tiles

## Manual Alternative

If you prefer to use an online favicon generator:

1. Use [RealFaviconGenerator](https://realfavicongenerator.net/)
2. Upload your source icon
3. Configure settings for all platforms
4. Download the generated files
5. Place them in the `public/` directory

## Notes

- The `favicon.ico` generated by the script is a simple copy of the 32x32 PNG. For a proper multi-size ICO file, use an online converter.
- SVG favicon will reference the PNG if source is not SVG format.
- All favicon links are already configured in `app/layout.tsx`.
